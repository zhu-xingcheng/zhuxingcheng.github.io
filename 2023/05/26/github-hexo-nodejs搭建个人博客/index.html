github-hexo-nodejs搭建个人博客

在github上搭建个人博客

预备

在github上新建仓库，仓库名为xxx.github.io

安装nodejs和git

安装hexo
npm install hexo-cli -g

本地ssh-keygen的公钥给到github

查看版本

部署博客

初始化博客目录

在博客目录 hexo init
若出现权限不够，按提示自行提权

如果报错Failed to install dependencies

运行npm install，可能会需要提权

运行

hexo generate
hexo server
在http://localhost:4000/ 打开博客demo

hexo d（上传）

写文章

文章在博客目录的source/_posts下
